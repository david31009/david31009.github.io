如何靠部落格流量賺廣告費 - 加入 Google Analytics 4 和 Google Adsense 吧

發表於2024-08-26|更新於2025-11-21|Hexo 系列文章
|閱讀量:

前言

除了用 Google Search Console 追蹤網站流量，最常用的應該就是 GA4 ( Google Analytics 4 )，GA4 是用來追蹤網站使用者數據的強大工具，只要安裝專屬的追蹤碼，就可以追蹤網站的流量、事件等數據。

加入 Google Analytics 4

  1. 申請 Google Analytics 帳戶

    • 新增資料串流: 登入 GA，點選左下角 管理> 資料收集與修改 > 資料串流 > 點選新增串流 > 網頁，接著把串流名稱 ( 可任意填 )、串流網址填好，點選新增

    • 獲取評估 ID: 新增完資料串流就會跳出評估 ID

  2. 修改 _config.butterfly.yaml， 把評估 ID ( G 開頭的一串代碼 ) 填到 # Google Analytics 區塊中

  3. 驗證

    • 按右鍵，檢查網頁原始碼，打開 chrome dev tool ( 開發工具 )，切到 Element 頁籤，搜尋 gtag

    • 在 GA4 後台測試是否有埋 GA4 的 tracking id

      • 新增串流時可以順便測試 ( 這個頁面先不要關 )，正確安裝會亮綠勾勾

      • 如果不小心關閉上面的測試頁面，可以到 資料串流 > 查看代碼操作說明 > 手動安裝

      • 點選測試: 出現綠勾勾代表成功埋入 ga4 的追蹤碼

加入 Google AdSense

加入 Google AdSense 的用意是：可以出租網站的部分版位給 google 放廣告，然後 google 會給你一些費用，但是聽說很少。基於好奇心，所以安裝看看。

  • 建立 Google AdSense 帳號

  • 完成三步驟 ( payments, ads, sites )

    1. payment 資訊: 之後 google 會付錢給你的帳戶，需要驗證手機號碼

    1. 設定 auto ads，自動生成廣告板位的預覽

    1. 加入 Google AdSense 的 client id: 也是一串 ID 要埋設在網站中

      • 修改 _config.butterfly.yaml，enable 設定成 true，新增 client id: ca-pub 開頭的代碼

      • 驗證

        • 查看網站原始碼: chrome dev tool 搜尋 adsbygoogle

        • 用 google ad sense 後台驗證

新增 ads.txt

  1. 用意: 列出哪些廣告商 ( 如 Google Adsense ) 有被我的網站授權。也就是說，有被我列出的廣告商，才可以用使我網站的版位投放 ( 賣 ) 廣告。

  2. 目的: 增強廣告市場的透明度和減少欺詐行為，也能夠賺取更多廣告收入。

    • 以下是來自 Google 官方說明 的文字:

      授權數位賣方 ( ads.txt ) 是一項 IAB Tech Lab 計劃，旨在協助您僅透過認定的授權賣方（如 AdSense）銷售您的數位廣告資源。

      建立自己的 ads.txt 檔案後，您可以更好地掌控哪些賣方有權銷售您網站上的廣告空間，並可防止向廣告客戶展示仿冒廣告資源。

      使用 ads.txt 並非強制性要求，但我們強烈建議您使用。它可以幫助買家識別仿冒廣告資源，並可以幫助您獲得更多廣告客戶支出，而這些支出原本可能會流向仿冒廣告資源

  • 到 Google Adsense 複製 ads.txt 內容。
    登入 Google Adsense > Site > 點選網站網址 > 複製內容

  • 在 source/ 下新增 ads.txt 檔

  • 部屬後，可以在 網域/ads.txt 檢查看看是否有新增成功
